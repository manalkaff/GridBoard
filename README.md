# GridBoard
Android Java Library


## Installation

First you need to add `jitpack` repositories
```
allprojects {
	repositories {
		...
		maven { url 'https://jitpack.io' }
	}
}
```
Then u can get the **GridBoard** library

```
dependencies {
	implementation 'com.github.manalkaff:GridBoard:Tag'
}
```
